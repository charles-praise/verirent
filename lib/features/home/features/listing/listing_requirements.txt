# VeriRent NG — Property Listing Requirements & Information Standards

## Essential Information for Property Advertisements

### 1. RESIDENTIAL PROPERTIES (House, Apartment, Duplex, Flat, Bungalow)

#### Core Information
- **Property Type**: House / Apartment / Duplex / Flat / Bungalow / Maisonette / Terrace
- **Listing Type**: For Rent / For Sale
- **Price**:
  - Rent: Annual amount (₦)
  - Sale: Full amount (₦)
  - Payment terms (lump sum, installments, etc.)
- **Location**:
  - Full street address
  - Area/District (e.g., GRA Phase 2, Trans Amadi)
  - LGA
  - State
  - GPS coordinates (optional but valuable)

#### Physical Specifications
- **Bedrooms**: Number (1BR, 2BR, 3BR, etc.)
- **Bathrooms**: Number
- **Toilets**: Number (if separate)
- **Living Spaces**:
  - Living room(s)
  - Dining room
  - Kitchen (fitted/unfitted)
  - Study/Office
  - Guest room
- **Area/Size**: Square meters (sqm) or plot size
- **Condition**: New / Recently Renovated / Good / Needs Renovation
- **Building Age**: Year built / Age of property

#### Amenities & Features
- **Furnishing**: Unfurnished / Part Furnished / Fully Furnished
- **Water Supply**: Borehole / Public supply / Both / Water tank
- **Power Supply**: NEPA / Generator / Solar / All
- **Air Conditioning**: Central AC / Split units / None
- **Parking**: Carport / Garage / Street parking / Number of spaces
- **Garden/Outdoor**: Garden / Balcony / Patio / Terrace
- **Security Features**: Fenced / Gated / Security guards / CCTV / Alarm system
- **Additional Features**:
  - Swimming pool
  - Gym/Fitness center
  - Laundry room
  - Storage room
  - Servant quarters
  - Maid's room
  - Store/Shop space

#### Documentation & Legal
- **Title Document Status**:
  - Certified deed / Government certificate / Lease document
  - Land use certificate
  - Survey plan reference
- **Registered/Verified**: Yes/No with certification level
- **Building Compliance**: Building approval / Completion certificate
- **Years Left on Lease** (if leasehold)

#### Neighborhood Information
- **Nearby Facilities**:
  - Schools (primary, secondary, university)
  - Hospitals/Clinics
  - Shopping centers
  - Markets
  - Banks
  - Place of worship
  - Recreational facilities
- **Transportation**: Distance to main roads, public transport access
- **Security Rating**: Area crime rate perception

#### Listing Details
- **Agency/Owner Details**:
  - Name
  - License/Verification status
  - Contact phone
  - Email
  - Office address
  - Years in business
  - Rating/Reviews
- **Listing Date**: When property was listed
- **Property Images**: Minimum 8-12 high-quality photos
  - Exterior views (front, side, back)
  - Living areas
  - Bedrooms
  - Bathrooms
  - Kitchen
  - Amenities
  - Neighborhood
- **Virtual Tour**: 360° photos or video tour
- **Description**: Professional summary highlighting key features
- **Inspection Availability**: Appointment booking system

## Additional Professional Standards

### Photography Standards
- Minimum 10-15 high-quality images per property
- Professional lighting (natural light preferred)
- Multiple angles of each major space
- Neighborhood context shots
- Clear, uncluttered composition
- Consistent color grading
- Current/recent photos (within 30 days)

### Description Guidelines
- Professional tone, 150-300 words
- Highlight unique selling points
- Specific amenities and features
- Location benefits
- Target audience identification
- Call-to-action

### Verification Badges
- ESVARBON Agency verification
- Individual verified status
- Property verification (document status)
- Years in business
- Client ratings
- Transaction history

### Legal Compliance
- Accurate pricing (no hidden charges disclosure)
- Legal document confirmation
- Clear lease/sale terms
- Dispute history (if any)
- Landlord/Owner consent confirmation

## VeriRent NG Quality Standards

All properties on VeriRent NG must meet the following:
1. **Agent/Owner Verification**: KYC completed
2. **Property Documentation**: Proof of ownership/authorization
3. **Accurate Information**: Price, size, amenities verified
4. **Professional Presentation**: Quality images and descriptions
5. **Contact Verification**: Active phone number and response time
6. **Regular Updates**: Re-listed every 30 days or marked unavailable
7. **Customer Reviews**: Available for completed transactions
